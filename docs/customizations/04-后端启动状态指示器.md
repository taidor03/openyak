# 04 - 后端启动状态指示器

> **定制编号**: XFLOW-014
> **涉及范围**: Rust 后端状态位 + Tauri IPC 事件驱动 + 前端 Provider + 指示器组件

---

## 一、功能概述

桌面端冷启动时，前端 UI 从本地缓存立即显示，但后端尚在启动（加载提供商、插件、MCP 等需约 5-30s），用户无法感知就绪状态。本模块实现了完整的后端启动状态指示与通知机制：

1. **Tauri IPC 事件驱动**：Rust 后端标记就绪状态 → 事件通知前端
2. **前端指示器组件**：三态 UI（启动中/就绪/同步完成）
3. **后端 URL 缓存**：就绪后自动缓存 URL

## 二、Rust 后端状态位

### 2.1 BackendState

**`desktop-tauri/src-tauri/src/backend.rs`**：

```rust
struct BackendInner {
    port: u16,
    process: Option<Child>,
    session_token: Option<String>,
    data_dir: Option<String>,
    backend_ready: bool,  // 后端就绪状态位
}

impl BackendState {
    pub async fn is_ready(&self) -> bool {
        let inner = self.inner.lock().await;
        inner.backend_ready
    }

    /// 启动后端进程
    pub async fn start(&self, app: &AppHandle) -> Result<String, String> {
        // 1. 标记 backend_ready = false
        // 2. 启动后端进程
        // 3. 后端启动成功后 → backend_ready = true
        // 4. 发送 "backend-ready" 事件通知前端
    }
}
```

### 2.2 IPC 命令注册

**`desktop-tauri/src-tauri/src/commands.rs`**：

```rust
#[tauri::command]
pub async fn is_backend_ready(state: tauri::State<'_, BackendState>) -> Result<bool, String> {
    Ok(state.is_ready().await)
}

#[tauri::command]
pub async fn get_backend_url(state: tauri::State<'_, BackendState>) -> Result<String, String> {
    let url = state.url().await;
    // Port 0 means the backend hasn't started yet — don't let the
    // frontend cache a dead URL that the /livez poll can't use.
    if url.ends_with(":0") {
        return Err("Backend not yet ready".to_string());
    }
    Ok(url)
}
```

### 2.3 窗口显示策略

**`desktop-tauri/src-tauri/src/lib.rs`**：

- **生产模式**：窗口立即显示，后端在后台启动
- 启动完成后通过 `backend-ready` 事件通知前端
- **开发模式**：后端已运行时立即标记 ready，跳过轮询

## 三、前端 BackendReadyProvider

### 3.1 Context Provider

**新增文件**：`frontend/src/components/providers/backend-ready-provider.tsx`（157 行）

```typescript
"use client";

import { createContext, useContext, useEffect, useState } from "react";

type Phase = "connecting" | "backend_ok" | "syncing" | "synced" | "done";

interface BackendReadyState {
  phase: Phase;
}

const BackendReadyContext = createContext<BackendReadyState>({ phase: "connecting" });

export function useBackendReady() {
  return useContext(BackendReadyContext);
}

export function BackendReadyProvider({ children }: { children: React.ReactNode }) {
  const [phase, setPhase] = useState<Phase>("connecting");

  useEffect(() => {
    // 1. IPC 就绪检查
    const checkIpcReady = async () => {
      try {
        const ready = await invoke<boolean>("is_backend_ready");
        if (ready) setBackendReady();
      } catch { /* not in Tauri */ }
    };

    // 2. 监听 Rust 后端事件
    const unlisten = listen("backend-ready", () => {
      setBackendReady();
    });

    checkIpcReady();
    return () => { unlisten.then(fn => fn()); };
  }, []);

  // Phase 流转
  // connecting → backend_ok (IPC事件) → syncing → synced → done
  // 完全事件驱动，无轮询降级
}
```

### 3.2 Phase 流转

```
connecting → backend_ok → syncing → synced → done
```

- **connecting**：后端未就绪
- **backend_ok**：IPC 事件通知后端就绪
- **syncing**：前端缓存同步中
- **synced**：缓存同步完成
- **done**：指示器隐藏

时间参数：
- `BACKEND_OK_DISPLAY_MS = 1s`
- `CACHE_SYNCED_DISPLAY_MS = 1.5s`

## 四、指示器组件

**新增文件**：`frontend/src/components/chat/backend-ready-indicator.tsx`

三态 UI，仅桌面模式渲染（Web/Remote 不显示）：

```tsx
"use client";

import { CheckCircle2 } from "lucide-react";
import { useTranslation } from "react-i18next";
import { useBackendReady } from "@/components/providers/backend-ready-provider";
import { IS_DESKTOP } from "@/lib/constants";

export function BackendReadyIndicator() {
  const { t } = useTranslation("chat");
  const { phase } = useBackendReady();

  if (!IS_DESKTOP || phase === "done") return null;

  // 状态 1: 启动中 — 橙色脉冲点
  if (phase === "connecting") {
    return (
      <div className="flex items-center gap-1.5 select-none" aria-live="polite">
        <span className="relative flex h-2 w-2 shrink-0">
          <span className="animate-ping absolute inline-flex h-full w-full rounded-full bg-amber-400 opacity-75" />
          <span className="relative inline-flex rounded-full h-2 w-2 bg-amber-500" />
        </span>
        <span className="text-[11px] font-medium text-[var(--text-tertiary)] whitespace-nowrap animate-pulse">
          {t("backendStartingService", { defaultValue: "正在启动服务" })}
        </span>
      </div>
    );
  }

  // syncing 阶段隐藏
  if (phase === "syncing") return null;

  // 状态 2: 就绪/同步完成 — 绿色勾
  const message =
    phase === "backend_ok"
      ? t("backendServiceStarted", { defaultValue: "后台服务启动成功" })
      : t("localCacheSynced", { defaultValue: "本地缓存已同步" });

  return (
    <div className="flex items-center gap-1.5 select-none animate-in fade-in duration-300" aria-live="polite">
      <CheckCircle2 className="h-3.5 w-3.5 text-emerald-500 shrink-0" />
      <span className="text-[11px] font-medium text-[var(--text-secondary)] whitespace-nowrap">
        {message}
      </span>
    </div>
  );
}
```

**挂载位置**：`frontend/src/components/chat/chat-header.tsx` ChatHeader 最右侧

**Provider 挂载**：`frontend/src/components/providers/app-providers.tsx`
