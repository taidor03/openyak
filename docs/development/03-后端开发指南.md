# OpenYak 后端开发指南

## 开发环境设置

### 前置要求

| 工具 | 版本 | 说明 |
|------|------|------|
| Python | 3.12+ | 必须，使用新语法特性 |
| pip | 最新 | 包管理 |
| Git | 2.x | 版本控制 |

### 安装依赖

```bash
cd backend
pip install -e ".[dev]"
```

> **注意**：`.[dev]` 会安装开发依赖（pytest、pytest-asyncio 等）。

### 配置环境变量

```bash
cp .env.example .env
# 至少配置一个 LLM 提供商的 API Key
# 例如: OPENYAK_OPENROUTER_API_KEY=sk-or-...
```

### 启动开发服务器

```bash
# 推荐（支持热重载）
uvicorn app.main:create_app --factory --reload --reload-dir app --host 0.0.0.0 --port 8000

# 或从项目根目录
npm run dev:backend
```

## 核心概念

### 会话处理流程（最重要）

后端最核心的代码在 `session/` 目录。理解 `SessionPrompt` 和 `SessionProcessor` 的分离是关键：

```
SessionPrompt（编排器 — 跨步骤状态）
  │
  ├── _setup()
  │   ├── 解析 agent, model, provider
  │   ├── 构建 system_prompt（agent + tools + memory + workspace）
  │   ├── 创建 UserMessage + UserPart
  │   └── 初始化跨步骤状态（cost, tokens, doom_history）
  │
  └── run() — Agent Loop
      │
      ├── while True:
      │   ├── processor = SessionProcessor(...)  # 每步新建
      │   ├── result = await processor()
      │   │   └── 返回 Continue | Stop | Compact
      │   │
      │   ├── if Continue: 继续循环
      │   ├── if Stop: break
      │   └── if Compact:
      │       └── await compact_context()
      │
      ├── 保存 AssistantMessage
      ├── 异步: 生成标题（title agent）
      ├── 异步: 计算摘要（summary agent）
      └── 异步: 提取记忆（memory updater）
```

### Agent 定义

Agent 在 `agent/agent.py` 中通过 `BUILTIN_AGENTS` 字典定义：

```python

        name="build",
        description="Full-featured AI assistant with all tools",
        mode="primary",

        permissions=Ruleset(rules=[
            PermissionRule(action="allow", permission="*"),
            PermissionRule(action="ask", permission="bash"),
            PermissionRule(action="allow", permission="code_execute"),
            PermissionRule(action="ask", permission="write"),
            PermissionRule(action="ask", permission="edit"),
            PermissionRule(action="allow", permission="plan"),
        ]),
        system_prompt=_load_prompt("build"),
    ),
    # ... 其他 agent
}
```

## API 开发

### 创建新路由

1. 在 `api/` 目录下创建新文件：

```python
# api/myfeature.py
from fastapi import APIRouter, Depends
from app.dependencies import get_settings

router = APIRouter(prefix="/api/myfeature", tags=["myfeature"])

@router.get("/items")
async def list_items(settings=Depends(get_settings)):
    return {"items": []}
```

2. 在 `api/router.py` 中注册：

```python
from app.api.myfeature import router as myfeature_router

api_router.include_router(myfeature_router)
```

### 依赖注入

使用 `dependencies.py` 中定义的依赖：

```python
from app.dependencies import (
    get_agent_registry,
    get_tool_registry,
    get_provider_registry,
    get_skill_registry,
    get_stream_manager,
)

@router.get("/tools")
async def list_tools(
    tool_registry=Depends(get_tool_registry)
):
    return tool_registry.list_tools()
```

## 数据库操作

### 定义模型

```python
from sqlalchemy import Column, String, Integer, Text
from app.models.base import Base, TimestampMixin
from app.models.base import generate_ulid

class MyModel(Base, TimestampMixin):
    __tablename__ = "my_table"

    id = Column(String, primary_key=True, default=generate_ulid)
    name = Column(String, nullable=False)
    data = Column(Text, nullable=True)  # JSON data 字段（常用模式）
```

### CRUD 操作

```python
from sqlalchemy import select, func
from sqlalchemy.ext.asyncio import AsyncSession

async def create_item(db: AsyncSession, name: str):
    item = MyModel(name=name)
    db.add(item)
    await db.commit()
    await db.refresh(item)
    return item

async def get_item(db: AsyncSession, item_id: str):
    result = await db.execute(select(MyModel).where(MyModel.id == item_id))
    return result.scalar_one_or_none()

async def count_items(db: AsyncSession):
    result = await db.execute(select(func.count()).select_from(MyModel))
    return result.scalar()
```

### 数据库迁移

```bash
# 创建迁移
alembic revision --autogenerate -m "add my_table"

# 应用迁移
alembic upgrade head
```

## 最佳实践

### 异步编程

- **始终使用 async/await**，避免同步阻塞
- 使用 `aiosqlite` 而不是 `sqlite3`
- 使用 `httpx` 而不是 `requests`
- 长时间操作使用 `asyncio.wait_for()` 超时控制

### 错误处理

```python
import logging
from fastapi import HTTPException

logger = logging.getLogger(__name__)

try:
    result = await some_operation()
except ValueError as e:
    logger.warning(f"Invalid input: {e}")
    raise HTTPException(status_code=400, detail=str(e))
except Exception as e:
    logger.error("Operation failed", exc_info=True)
    raise HTTPException(status_code=500, detail="Internal error")
```

新增配置项在 `config.py` 的 `Settings` 类中添加：

```python
class Settings(BaseSettings):
```


## 常见问题

### 如何添加新的环境变量？

在 `config.py` 的 `Settings` 类中添加字段，环境变量名自动映射为 `OPENYAK_{FIELD_NAME}`。

### 如何添加新的数据库表？

1. 在 `models/` 中定义模型
2. 创建迁移：`alembic revision --autogenerate -m "add my_table"`
3. 应用迁移：`alembic upgrade head`

### 如何调试 SSE 流？

```bash
curl -N http://localhost:8000/api/chat/stream/{stream_id}
```

### 如何添加新的中间件？

在 `main.py` 的 `create_app()` 中添加：

```python
app.add_middleware(MyMiddleware)
```
