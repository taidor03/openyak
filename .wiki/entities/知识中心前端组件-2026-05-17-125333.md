# 知识中心前端组件 — KnowledgeCenterContent

`KnowledgeCenterContent` 是知识中心的主要 React 组件，定义在 `frontend/src/app/(main)/knowledge/content.tsx`，约 1058 行。它是页面的核心功能层，包含页面浏览、编辑、搜索、分类过滤等完整 UI 逻辑。

useEffect(() => {
  setSearchResults([]);
  setSelectedCategory(null);
  setSearchQuery("");
  if (!showEditor) setSelectedPage(null);

  const init = async () => {
    setIsLoading(true);
    const data = await fetchStatus();
    if (data && !data.initialized) {
      await api.post(wikiUrl("/initialize"));
      await fetchStatus();
    }
    await fetchPages();
    setIsLoading(false);
  };
  init();
}, [target, fetchStatus, fetchPages, wikiUrl]);

### 5. Wikilink 预处理与渲染

```typescript
// 预处理：[[target|alias]] → [alias](wiki:target)
function preprocessWikiMarkdown(raw: string): string {
  const content = raw.replace(/^---\n[\s\S]*?---\n/, "");
  return content.replace(
    /\[\[([^\]|]+?)(?:\|([^\]]+))?\]\]/g,
    (_, target, alias) => `[${alias || target}](wiki:${target.trim()})`
  );
}

// ReactMarkdown 自定义 a 标签渲染
const components = useMemo(() => ({
  a: ({ children, href, ...props }) => {
    if (href?.startsWith("wiki:")) {
      const target = decodeURIComponent(href.slice(5));
      return (
        <span className="text-[var(--brand-primary)] underline ..."
              role="button"
              onClick={() => onWikilinkClick?.(target)}>
          {children}
        </span>
      );
    }
    return <a target="_blank" href={href} {...props}>{children}</a>;
  }
}), [onWikilinkClick]);
```

  const base = `/api/wiki${path}`;

## 已知设计要点

| 要点 | 说明 |
|---|---|
| **全量重载策略** | 编辑模式下不会增量更新，而是 `POST` 写入后 `fetchPages()` 全量刷新 |
| **防抖搜索不取消** | 在 300ms 内发起多次请求时，旧的请求结果仍可能晚到并被采纳；关闭编辑时取消但不全面 |
| **编辑器关闭后不清空编辑状态** | `showEditor` 变化时不触发重新初始化 effect |
| **merge 后端逻辑** | `/merge` 接口当前查找同名页面（未支持 `sections` 参数），前端传的 `sections` 字段实际不被后端使用 |
| **无 TanStack Query 缓存** | 所有数据通过 `fetchStatus`/`fetchPages` 手动管理，无持久化 |
| **无国际化覆盖** | 搜索等异步操作的 Toast 消息使用 `t("searchFailed")` 但搜索输入框标签是硬编码英文 |
